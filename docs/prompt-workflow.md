# Prompt 工作流文档

## 1. AI 学习助手核心工作流

EduAI Copilot 的核心是 5 个学习场景，每个场景都有对应的 Prompt 模板。

工作流：
```
学生输入问题 → AI 识别场景 → 调用对应 Prompt → 生成回答 → 学生追问/确认
```

5 个核心场景：
1. 课程资料总结
2. 概念解释
3. 复习计划生成
4. 练习题生成
5. 学习反馈分析

## 2. 课程资料总结 Prompt

### 场景说明

学生上完课，把零散的笔记丢给 AI，AI 帮他整理成结构化的知识总结。

### Prompt 模板

```
你是一位学习助手。请将以下课堂笔记整理成结构化的知识总结。

要求：
1. 提取核心概念
2. 整理关键公式或要点
3. 说明实际应用场景
4. 标注重点和难点
5. 给出学习建议

课堂笔记：
{学生输入}

请用 Markdown 格式输出。
```

### 输入示例

```
今天学了导数，导数就是函数变化的速度，用极限来算。几何意义是切线斜率。还讲了几个求导公式，x^n 的导数是 nx^(n-1)，sinx 的导数是 cosx。
```

### 输出示例

```markdown
## 导数知识总结

### 核心概念
- 导数描述函数在某一点的变化速度
- 用极限定义：f'(x) = lim(Δx→0) [f(x+Δx) - f(x)] / Δx
- 几何意义：曲线在该点的切线斜率

### 关键公式
- (x^n)' = nx^(n-1)
- (sin x)' = cos x
- (cos x)' = -sin x

### 应用场景
- 求函数的极值（最大值最小值）
- 分析函数的单调性（什么时候递增什么时候递减）
- 解决实际问题（比如速度、加速度）

### 重点难点
- 重点：导数的定义和几何意义
- 难点：复合函数求导（链式法则）

### 学习建议
- 先理解导数的几何意义，再记公式
- 多做求导练习，熟练掌握常见函数的导数
- 结合图形理解，画图帮助记忆
```

### 设计思路

- 让 AI 提取笔记中的关键信息
- 用结构化格式整理，方便复习
- 标注重难点，帮助学生抓住重点
- 给出学习建议，引导下一步学习

## 3. 概念解释 Prompt

### 场景说明

学生遇到不懂的概念，问 AI，AI 用通俗语言解释清楚。

### Prompt 模板

```
你是一位学习导师。请解释以下概念，让学生真正理解。

要求：
1. 用通俗类比解释（像给小学生讲一样）
2. 给出准确的技术定义
3. 举一个实际应用的例子
4. 说明常见的理解误区

概念：{学生输入}

请用简单易懂的语言解释。
```

### 输入示例

```
什么是"过拟合"？
```

### 输出示例

```markdown
## 过拟合解释

### 通俗类比
就像学生只背答案不理解原理，考试遇到新题就不会了。

### 技术定义
模型在训练数据上表现很好，但在新数据上表现差。

### 实际例子
训练一个识别猫的模型，如果只用家猫的图片训练，模型可能把所有长得像家猫的都认成猫，但遇到野猫就认不出来了。这就是过拟合。

### 常见误区
- 误区：模型越复杂越好
- 真相：模型太复杂容易过拟合，要在复杂度和泛化能力之间找平衡

### 怎么避免
- 增加训练数据的多样性
- 使用正则化技术
- 交叉验证
```

### 设计思路

- 用通俗类比降低理解门槛
- 给准确定义建立正确认知
- 用实际例子帮助理解
- 指出常见误区避免踩坑

## 4. 复习计划 Prompt

### 场景说明

学生要考试了，告诉 AI 考试信息，AI 帮他安排复习计划。

### Prompt 模板

```
你是一位学习规划师。请根据以下信息制定复习计划。

考试信息：
- 考试科目：{科目}
- 考试时间：{时间}
- 要复习的内容：{内容}
- 每天可用时间：{小时}

要求：
1. 合理分配每天的复习内容
2. 明确每天的学习目标
3. 标注重难点
4. 给出复习建议

请输出每日复习计划。
```

### 输入示例

```
考试科目：高等数学
考试时间：下周六
要复习的内容：函数、极限、导数、积分
每天可用时间：3 小时
```

### 输出示例

```markdown
## 高数复习计划

### Day 1：函数与极限（3 小时）
- 上午：复习函数概念和性质（1.5 小时）
  - 重点：函数的定义域、值域、单调性
  - 做课后习题 1-10
- 下午：复习极限（1.5 小时）
  - 重点：极限的定义和计算方法
  - 做课后习题 11-20

### Day 2：导数（3 小时）
- 上午：复习导数定义和公式（1.5 小时）
  - 重点：导数的几何意义、常见函数导数
  - 做课后习题 21-30
- 下午：复习求导法则（1.5 小时）
  - 重点：链式法则、乘积法则
  - 做课后习题 31-40

### Day 3：积分（3 小时）
- 上午：复习不定积分（1.5 小时）
  - 重点：基本积分公式、换元法
  - 做课后习题 41-50
- 下午：复习定积分（1.5 小时）
  - 重点：定积分的计算和应用
  - 做课后习题 51-60

### Day 4：综合复习（3 小时）
- 上午：做模拟题（2 小时）
- 下午：整理错题（1 小时）

### Day 5：重点突破（3 小时）
- 根据模拟题情况，重点复习薄弱环节

### 复习建议
- 每天先看笔记，再做题
- 遇到不会的先自己想，实在不会再看答案
- 整理错题，考前重点看
```

### 设计思路

- 根据考试时间合理分配任务
- 每天有明确的学习目标
- 标注重难点，抓住重点
- 给出复习建议，提高效率

## 5. 练习题生成 Prompt

### 场景说明

学生想巩固某个知识点，让 AI 生成练习题。

### Prompt 模板

```
你是一位出题老师。请根据以下知识点生成练习题。

知识点：{学生输入}
难度：{难度}
题目数量：{数量}

要求：
1. 题目类型多样（选择、填空、解答）
2. 难度适中，不要太难也不要太简单
3. 覆盖知识点的不同方面
4. 给出详细解析

请生成练习题。
```

### 输入示例

```
知识点：导数的计算
难度：中等
题目数量：5 道
```

### 输出示例

```markdown
## 导数计算练习题

### 题目 1（选择题）
函数 f(x) = x³ 的导数是：
A. x²  B. 3x²  C. 3x  D. 3x³

**答案：B**
**解析**：使用幂函数求导公式 (x^n)' = nx^(n-1)，这里 n=3，所以 f'(x) = 3x²。

### 题目 2（填空题）
函数 f(x) = sin x 的导数是 ______。

**答案：cos x**
**解析**：sin x 的导数是 cos x，这是基本导数公式。

### 题目 3（解答题）
求函数 f(x) = x² + 2x + 1 的导数。

**答案**：f'(x) = 2x + 2
**解析**：使用求导法则，分别对每一项求导：
- (x²)' = 2x
- (2x)' = 2
- (1)' = 0
所以 f'(x) = 2x + 2。

### 题目 4（选择题）
函数 f(x) = eˣ 的导数是：
A. eˣ  B. xeˣ⁻¹  C. eˣ⁻¹  D. 1

**答案：A**
**解析**：eˣ 的导数还是 eˣ，这是指数函数的特殊性质。

### 题目 5（解答题）
求函数 f(x) = x · sin x 的导数。

**答案**：f'(x) = sin x + x · cos x
**解析**：使用乘积法则 (uv)' = u'v + uv'：
- u = x, u' = 1
- v = sin x, v' = cos x
所以 f'(x) = 1 · sin x + x · cos x = sin x + x · cos x。
```

### 设计思路

- 题目类型多样，全面考察
- 难度适中，适合巩固
- 详细解析，帮助理解
- 覆盖知识点不同方面

## 6. 学习反馈分析 Prompt

### 场景说明

学生做完题或考完试，把错题给 AI，AI 分析错因并给出建议。

### Prompt 模板

```
你是一位学习分析师。请分析以下错题，找出错因并给出改进建议。

错题信息：
{学生输入}

要求：
1. 分析每道题的错因
2. 找出知识薄弱点
3. 给出针对性的学习建议
4. 推荐巩固练习

请给出详细分析。
```

### 输入示例

```
错题 1：求 f(x) = x³ 的导数，我写成了 3x
错题 2：求 f(x) = sin x 的导数，我写成了 -cos x
错题 3：求 f(x) = x² · eˣ 的导数，我不会做
```

### 输出示例

```markdown
## 错题分析报告

### 错题 1 分析
**题目**：求 f(x) = x³ 的导数
**你的答案**：3x
**正确答案**：3x²

**错因分析**：
- 幂函数求导公式记错了
- 应该是 nx^(n-1)，你写成了 nx^(1)

**知识薄弱点**：幂函数求导公式

### 错题 2 分析
**题目**：求 f(x) = sin x 的导数
**你的答案**：-cos x
**正确答案**：cos x

**错因分析**：
- 基本导数公式记混了
- sin x 的导数是 cos x，不是 -cos x
- -cos x 是 cos x 的导数

**知识薄弱点**：三角函数导数公式

### 错题 3 分析
**题目**：求 f(x) = x² · eˣ 的导数
**你的答案**：不会

**错因分析**：
- 不会用乘积法则
- 这是两个函数相乘，需要用 (uv)' = u'v + uv'

**知识薄弱点**：乘积法则

### 整体分析

**主要问题**：
1. 基础公式记忆不牢
2. 求导法则不熟练

**学习建议**：
1. 重新背诵基本导数公式（每天默写一遍）
2. 专项练习乘积法则和链式法则
3. 做完题后检查一遍，避免计算错误

**巩固练习**：
- 基础公式：求 f(x) = cos x, f(x) = eˣ, f(x) = ln x 的导数
- 乘积法则：求 f(x) = x · cos x, f(x) = x² · sin x 的导数
- 链式法则：求 f(x) = sin(2x), f(x) = e^(x²) 的导数
```

### 设计思路

- 逐题分析错因，精准定位问题
- 找出知识薄弱点，针对性提升
- 给出具体的学习建议
- 推荐巩固练习，帮助提高

## 7. Prompt 设计原则

### 原则 1：角色明确

每个 Prompt 都给 AI 一个明确的角色：
- 学习助手：整理笔记
- 学习导师：解释概念
- 学习规划师：制定计划
- 出题老师：生成练习题
- 学习分析师：分析错题

### 原则 2：要求具体

每个 Prompt 都给出具体的要求：
- 要做什么
- 输出什么格式
- 包含哪些内容

### 原则 3：引导思考

不直接给答案，而是引导学生思考：
- 解释为什么
- 给出思考过程
- 提供学习建议

### 原则 4：结构化输出

输出格式清晰，方便阅读：
- 使用 Markdown 格式
- 分点说明
- 标注重难点
